Description

Pak Dengklek reminisces about the time when he first started raising ducks.

At first, Pak Dengklek did not have any ducks. Every day for $N$ consecutive days, he adopted a new duck. More specifically, at the end of the $i$-th day, Pak Dengklek adopted a duck that is $A_i$ days old. Note that the ages of the ducks that Pak Dengklek already owns will increase with each passing day.

Pak Dengklek is curious, at the end of each day, what is the age of the youngest duck, the oldest duck, and the average age of all his ducks?

Constraints

  • $1 \leq N \leq 100\,000$
  • $1 \leq A_i \leq 200\,000$

Subtasks

  1. (10 points) Contains only the following test case: 
    5
4 2 3 9 4
  2. (16 points) $A_i = i$
  3. (12 points) $A_1 = A_2 = \ldots = A_N$
  4. (32 points) $N \leq 1000$
  5. (30 points) No additional constraints.

For each subtask, you can obtain $50\%$ of the subtask points if you can only answer the youngest age and the oldest age correctly for each day (and not answer the average age correctly).

Input

The input is given in the following format: 

N
A1 A2 … AN

Output

Output $N$ rows, each consisting of two integers and a real number representing the age of the youngest duck, the age of the oldest duck, and the average age of all ducks respectively. The average age will be considered correct if its absolute difference with the judge's output is less than $10^{-2}$. Note that you must always output three numbers on each line, even if you intend to only obtain $50\%$ points.

Sample Input

4
3 1 3 7

Sample Output

3 3 3.0000
1 4 2.5000
2 5 3.3333
3 7 5.0000

Explanation of Sample

  • On the $1$-st day, Pak Dengklek's duck is $3$ days old.
  • On the $2$-nd day, Pak Dengklek's ducks are $4$ and $1$ days old.
  • On the $3$-rd day, Pak Dengklek's ducks are $5$, $2$, and $3$ days old.
  • On the $4$-th day, Pak Dengklek's ducks are $6$, $3$, $4$, and $7$ days old.
