Description

There are $N$ plots of land numbered from $1$ to $N$ arranged in a circle. Plot $i$ is immediately to the left of plot $i + 1$ for each $1 \leq i \leq N - 1$, and plot $N$ is immediately to the left of plot $1$.

Bu Dengklek knows that exactly one of the $N$ plots contains a treasure and she wants to find the location of that plot. To assist her search, on each plot $i$, Bu Dengklek installed a machine numbered $i$.

In one operation, Bu Dengklek can choose an integer $k$ and activates $k$ different machines. After that, Bu Dengklek will obtain information about the shortest distance between the nearest active machine and the treasure. This operation costs $\left\lceil\frac{N}{k}\right\rceil$ Kwakkoins. After the operation, all active machines are deactivated again.

Bu Dengklek has only $512$ Kwakkoins. Help Bu Dengklek find the location of the plot containing the treasure!

Note:

  • Formally, the shortest distance between plot $x$ and plot $y$ ($x \leq y$) is the minimum of $y - x$ and $N - (y - x)$.
  • $\lceil w\rceil$ for a real number $w$ denotes the value of $w$ rounded up.

Constraints

  • $N = 256$

Subtasks

  1.   (9 points) The treasure is located on plot $1$.
  2. (15 points) The treasure is located on one of the plots $1$ through $3$ inclusive.
  3. (33 points) The treasure is located on one of the plots $1$ through $128$ inclusive.
  4. (43 points) No additional constraints.

Problem Type Information

This is an "interactive" problem. In this problem, you will interact with the judge's program through the standard input (stdin) and the standard output (stdout). See the following interaction format carefully.

Interaction Format

To perform an operation that activates $k$ machines $a_1$, $a_2$, ..., $a_k$ ($1 \leq a_1 < a_2 < \ldots < a_K \leq N$), you can output to the standard output in the following format: 

TANYA k
a1 a2 … ak

Note that the machine numbers must be strictly increasing.

For each operation, you will be given the answer from the standard input in the following format: 

d

This means, the shortest distance between an active machine and the treasure is $d$.

If you have located the treasure, for example on plot $z$, you can output to the standard output in the following format: 

JAWAB z

Scoring

Each operation that activates $k$ machines costs $\left\lceil\frac{N}{k}\right\rceil$ Kwakkoins. Let $C$ denote the total Kwakkoins over all operations. A $\text{PERFORMA}$ score is computed according to the following table:

Condition $\text{PERFORMA}$
$512 < C$ $0\%$
    $6 < C \leq 512$ $(100 - 5 \times \log_2(C - 5))\%$
$C \leq 6$     $100\%$

For each test case, its $\text{PERFORMA}$ score is computed. Each subtask consists of multiple test cases. For each subtask, the smallest $\text{PERFORMA}$ score will be taken. The score for a subtask is the product of the smallest $\text{PERFORMA}$ score and the subtask points. The final score is the sum of the scores of all subtasks, which is then rounded down.

Your program will receive "Wrong Answer" if it:

  • uses more than $512$ Kwakkoins, or
  • produces an output in a wrong format or outside of the constraints.

If a test case is successfully executed, you can see its detailed scoring in the submission page. You can also view the total Kwakkoins used for that test case.

The grader is not adaptive. In other words, the location of the plot that contains the treasure is fixed before the interaction begins.

Sample Interaction

The following is a possible sample interaction. In this example, the treasure is located on plot $42$.

Input Output Explanation 
TANYA 3
2 5 6
This operation activates machines $2$, $5$, and $6$. This operation costs $\left\lceil\frac{256}{3}\right\rceil = 86$ Kwakkoins. 
36
 The active machine that is closest to the treasure is machine $6$ with distance $6$. 
TANYA 2
196 234
This operation activates machines $196$ and $234$. This operation costs $\left\lceil\frac{256}{2}\right\rceil = 128$ Kwakkoins. 
64
 The active machine that is closest to the treasure is machine $234$ with distance $64$. 
TANYA 4
26 52 78 104
This operation activates machines $26$, $52$, $78$, and $104$. This operation costs $\left\lceil\frac{256}{4}\right\rceil = 64$ Kwakkoins. 
10
 The active machine that is closest to the treasure is machine $52$ with distance $10$. 
JAWAB 42
 Answering that the treasure is on plot $42$.
(end) (end)

In the sample above, the total Kwakkoins used is $86 + 128 + 64 = 278$. Since $6 < 278 \leq 512$ holds, then the $\text{PERFORMA}$ score is $(100 - 5 \times \log_2(278 - 5))\% \approx 59.536\%$.

Warning

Always flush after each output.

  • For stdio.h(cstdio) library: 
    fflush(stdout);
  • For iostream library: 
    std::cout << std::flush;
