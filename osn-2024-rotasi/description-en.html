Description

Si Pitung is leading his troops to defend Batavia. There are $N$ soldiers lined up in one line. Initially, the soldier at position $i$ from the front has power $A_i$.

Si Pitung can carry out two types of commands:

  • Rotation command: command the soldier at the end of the line to move to the front of the line.
  • Extension command: add a new soldier with any power to the end of the line.

Si Pitung has an integer $K$. For a line consisting of $c$ soldiers, an interval $[x, y]$ ($1 \leq x \leq y \leq c$) is solid if and only if both of these requirements are satisfied:

  • $y - x + 1 = K$, and
  • for each integer $p$ from $x$ to $y$, there is a soldier with power $p$ at position $q$ from the front of the line, satisfying $x \leq q \leq y$.

Si Pitung would like to carry out zero or more commands so his soldier lineup has at least one solid interval. Rotation commands can be carried out as many times as Si Pitung wants, however extension commands require more planning to be carried out. What is the minimum number of extension commands that has to be carried out by Si Pitung so that his soldier lineup has at least one solid interval?

Constraints

  • $1 \leq K \leq N \leq 300\,000$
  • $1 \leq A_i \leq N$

Subtasks

  1.   (5 points) Contains only the following test case:
    10 4
1 7 8 4 5 1 7 8 2 4
  2. (13 points) $K \leq 3$
  3. (20 points) $N \leq 50$
  4. (21 points) $N \leq 3000$
  5. (20 points) $N \leq 80\,000$
  6. (21 points) No additional constraints.

Input

The input is given with the following format: 

N K
A1 A2 … AN

Output

Output an integer representing the minimum number of extension commands that has to be carried out by Si Pitung so that his soldier lineup has at least one solid interval.

Sample Input

4 3
1 4 4 1

Sample Output

2

Explanation of Sample

Si Pitung can carry out the following commands, consisting of $2$ extension commands:

  1. Rotation command. The lineup becomes $[1, 1, 4, 4]$.
  2. Extension command for a soldier of power $3$. The lineup becomes $[1, 1, 4, 4, 3]$.
  3. Rotation command. The lineup becomes $[3, 1, 1, 4, 4]$.
  4. Rotation command. The lineup becomes $[4, 3, 1, 1, 4]$.
  5. Extension command for a soldier of power $5$. The lineup becomes $[4, 3, 1, 1, 4, 5]$.
  6. Rotation command. The lineup becomes $[5, 4, 3, 1, 1, 4]$.
  7. Rotation command. The lineup becomes $[4, 5, 4, 3, 1, 1]$.
  8. Rotation command. The lineup becomes $[1, 4, 5, 4, 3, 1]$.

After carrying out all the commands above, the interval $[3, 5]$ is a solid interval since the following conditions are satisfied:

  • There is a soldier with power $3$ at position $5$ ($3 \leq 5 \leq 5$).
  • There is a soldier with power $4$ at position $4$ ($3 \leq 5 \leq 5$).
  • There is a soldier with power $5$ at position $3$ ($3 \leq 5 \leq 5$).
